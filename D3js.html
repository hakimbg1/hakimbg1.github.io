Description

D3.js is a JavaScript graphics library that allows the display of numerical data in a graphical and dynamic form. It is an important tool for W3C compliance, using common technologies such as SVG, JavaScript, and CSS for data visualization.

Our professor assigned each of us a discussion topic to learn the D3.js library and use it for this purpose.

I divided the project into 4 main parts:


  • Download Data: For my project, I chose the evolution of the coronavirus in 7 countries and stored the data in a CSV file. Using the "console log(data)" command, I noticed that each piece of data was stored as a string, so I had to define the type of each line.

  • Using D3.nest: The goal is to group each country into a single line and aggregate all the points for that country.

  • Displaying Multiple Lines: To display multiple lines, we can use a data join.

  • Different Colors for Each Line: For this, I can't simply use a CSS file, or I would have a single color for all lines. So, I had to find a solution, and I discovered that this can be done using "colorScale" and choosing from a palette of multiple colors.

Conclusion

Working with D3.js was a challenge for me, not only because of the deadline but also because I learned a skill on my own while adhering to what I could and couldn't do in a short time. Even though I consider this library not to be the only solution, as there are multiple easy and flexible solutions, the goal was to explore the variety of solutions to build a graphical view on the web, and this library does its job very well.
